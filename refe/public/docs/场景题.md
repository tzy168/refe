### 2. 如何解决页面请求接口大规模并发？

1. 滑动窗口算法，专门用来控制流量
2. 背景：数据采集，低代码，有序相对稳定发送后端
3. 自己封装请求队列 queue = [ ]
4. 防抖节流 **lodash**

- 防抖： 最后一次触发，延迟执行
- 节流： 一段时间内，只执行一次

5. 分页、滚动加载 **可视区域绘制**
